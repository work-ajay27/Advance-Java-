# 🎓 Student Management System

This project implements a **Student Management System** in Java, allowing users to manage student records efficiently.

## 📌 Features
- Add a new student
- View all students
- Search for a student by ID
- Update student details
- Delete a student by ID

## 🛠 Technologies Used
- **Java**

## 🎯 Purpose of the Code
- Demonstrates CRUD operations on student records.
- Provides a structured way to manage student data.

## 📄 Notes
- This is a foundational project to understand object-oriented programming concepts.
